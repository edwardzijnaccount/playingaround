## Installation
Install the dependencies

```sh
git clone https://github.com/edwardzijnaccount/playingaround.git
# change the .py files
git add .
git commit .

```

Structure of project set up

```sh
#within git

    README.md

    app.py 
    config\
        config.json
        config.yaml
    lib\
        DataConfig.py
        ImportConfData.py
        ImportData.py
        __init__.py
```
